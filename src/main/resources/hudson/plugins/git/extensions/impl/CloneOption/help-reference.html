Specify a directory path to a local mirror that will be used during clone operations, e.g. /opt/my git mirror/some-project.git
Variables in the form $VARIABLE or ${VARIABLE} will be expanded as appropriate at build time.

When a valid path is specified, cloning from a "reference" repository should be faster and consume less disk space than cloning solely from the configured remote repository URL.
See the git clone --reference documentation for more information.

This option will be ignored if the given directory does not exist on the machine where the clone operation is being executed.
